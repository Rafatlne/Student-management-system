1. Put the folder in C:\xampp\htdocs.
2. Open phpmyadmin on your default browser.
3. Create a new database. Name will be "student" without "".
4. Click on that database and import a student.sql file which included in the zip. You will find it on "Project\theme\databaseFile" folder.
5. Now paste this url = "localhost/Project" in your browser addressbar.
